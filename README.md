Salvador is a pixel-based graphics editor built as a university project for **Computer Graphics** course at **Bialystok University of Technology**. Named after the legendary artist **Salvador Dal√≠**, it aims to bring professional image editing capabilities directly to your browser.

## Tech Stack

- **Next.js** - React framework
- **TypeScript** - Type-safe development
- **Canvas** - Low-level pixel manipulation
- **Tailwind CSS** - Styling

## Getting Started

```bash
npm install
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to start creating!
